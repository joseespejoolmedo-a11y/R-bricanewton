📥 Importar datos de la rúbrica EF

Pega aquí el JSON completo que exportaste. Antes de importar, esta página hará una copia de seguridad automática de los datos actuales en backup_rubrica_....

Este importador sirve para pasar datos antiguos como rubricaEF_db_v3 a la rúbrica nueva que usa rubrica_ef_editable_db_v1.

Uso rápido

1. Pega el JSON exportado.
2. Pulsa Importar datos.
3. Después pulsa Abrir rúbrica.

Si solo quieres transformar la base antigua rubricaEF_db_v3 en la nueva rubrica_ef_editable_db_v1, pulsa Convertir v3 a nueva rúbrica.
